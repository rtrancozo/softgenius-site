# SoftGenius Site

Site institucional da SoftGenius, empresa de tecnologia focada em desenvolvimento de softwares, sistemas SaaS, automação, soluções digitais, ensino e treinamento em tecnologia.

## Como executar

Instale as dependências e rode o projeto localmente.

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Observações

Substitua os arquivos em `public/assets` pelas logomarcas oficiais quando estiverem finalizadas.
